== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `FCF-Pay` and click 'Install Now' or click on the `upload` link to upload `fcf-pay.zip`
4. Click on `Activate plugin`
